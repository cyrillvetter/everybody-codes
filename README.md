# Everybody Codes

This repository contains my personal solutions to the Everybody Codes puzzles, written in Haskell.

## Puzzle Inputs

**The puzzle inputs are not included in this repository.**

Each user receives their own input data when solving puzzles on the Everybody Codes website. Because these inputs are tied to user accounts and distributed by the event organizers, they are not stored in this repository.

## Running a Solution

If you have the Haskell toolchain installed, you can run a solution with:

```
cd 2024
cabal run 01
```
